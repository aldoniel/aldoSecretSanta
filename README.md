# aldoSecretSanta

## Intro

* Un tireur au sort de Secret Santa respectueux des données.
* Application en page web unique.
* Partagez le tirage au sort du Secret Santa par une url que l'app. génère.
* Un auteur crée la liste des noms, et partage l'url aux utilisateurs.
* Ne demande aucune inscription, aucune information personnelle. Toutes les informations sont dans l'url à partager. (Le serveur n'enregistre rien, tout est traité dans le navigateur.).
* Personne ne connait le tirage au sort, pas même le créateur.

## Accès
*  [Cliquez ici pour l'application en ligne](https://aldoniel.github.io/aldoSecretSanta/)

## Technique
* Code Brython 3.11
* Utilise du code IA pour frenchnamecompressor.py qui compresse les url. (la compression est basée sur un index des 300 prénoms Insee les plus fréquents).
